    Python Example: Squaring Numbers #

    This is a simple Python script that demonstrates how to square each number in a list using a function and list comprehension.

    Code Snippet # 

    # Square numbers in a list and print the results

def square_numbers(numbers):
    """Return a list of squared numbers."""
    return [x**2 for x in numbers]

# Example usage
input_numbers = [1, 2, 3, 4, 5]
squared = square_numbers(input_numbers)

print("Original numbers:", input_numbers)
print("Squared numbers:", squared)

# Output

Original numbers: [1, 2, 3, 4, 5]
Squared numbers: [1, 4, 9, 16, 25]
